About Car Price Prediction with Configurator

I am working on Automotive Industry . My project Title is Car Price Prediction with Configurator Demo. The focus of this project is developing machine learning models that can accurately predict the price of a used car based on its features, in order to make informed purchases.Our results show that Random Forest model yield the best results . Deciding whether a used car is worth the posted price when you see listings online can be difficult. For this project, we are using the dataset on used car sales from all over India. More Than this I also used pixel streaming in this project which makes my project more interesting and interactive with 3-D application. This gives user the experience as if they were using the application right on their own computer or mobile device. Some common use cases for this are for games, simulations, or immersive experiences such as configuring a car or walking through a virtual showroom.
In this Project I used Following techniques and tools -

  • Primary Azure Technology
  • Virtual Machines
  • Azure Functions
  • Web Development (HTML , CSS , Flask)
  • Machine Learning
  • Visual Studio Code
  • Power BI For Data Analysis
  • Azure For Deployment
The codes for this project can be checked in my github repo.
